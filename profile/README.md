# botzone-ai

Open-source SDKs for [Botzone](https://botzone.ai) products. Wrap your LLM clients in one line, attribute every euro to a route or feature, see where the money goes.

## Repositories

| Repo | Package | Language |
|------|---------|----------|
| [cost-sdk-ts](https://github.com/botzone-ai/cost-sdk-ts) | [`@botzone/cost-sdk`](https://www.npmjs.com/package/@botzone/cost-sdk) | TypeScript |
| [cost-sdk-py](https://github.com/botzone-ai/cost-sdk-py) | [`botzone-cost`](https://pypi.org/project/botzone-cost/) | Python |

## Products

- **[Cost](https://cost.botzone.ai)** — LLM cost attribution and observability. The SDKs in this org ship metadata-only events to the Cost dashboard, which surfaces per-route spend, eval-gated downgrade recommendations, and verified savings.
